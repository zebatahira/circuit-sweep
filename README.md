# Circuit Sweep

A neon-themed Minesweeper game built with plain HTML, CSS, and JavaScript.

## How to play

- Choose a difficulty: Beginner, Intermediate, or Expert.
- Left click a cell to reveal it. Your first click is always safe.
- Numbers show how many mines are in the surrounding 8 cells.
- Reveal all safe cells to win. Click a mine and it's game over.
- Click the smiley to restart. Click the `i` button for full instructions.

## Tech

HTML, CSS, and vanilla JavaScript — no frameworks, no build step. Everything is in one file: `index.html`.
